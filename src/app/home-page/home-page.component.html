    Ιεραρχία:
  • Home component -> χρησιμοποιούμε το service για να φέρουμε τα data όλων των players. Στο template του εμφανίζουμε το table component στο οποίο με data binding περνάμε την λίστα των players.
  • Player table component -> λαμβάνουμε τα data των players και στο template του τα εμφανίζουμε σε kendo grid. Με *ngIf εμφανίζουμε ή κρύβουμε το tabs component. Έχουμε προσθέσει λειτουργικότητα κλικ που στέλνει στο tabs component το object.
  • Tabs component -> λαμβάνουμε το οbject του player το οποίο το παίρνουμε και το περνάμε στο fiels component. Απλά κουβαλάει το object και έχει την λειτουργικότητα της kendo tabs.ώστε να εμφανίσουμε τα στοιχεία του σε disabled ή enabled πεδία. Τα πεδία τα φέρνουμε από το fields component.
  • Fiels component -> λαμβάνουμε το οbject του player το οποίο το παίρνουμε και το περνάμε στα πεδία της φόρμας. Υπάρχει button το οποίο ενημερώνει το object με νέο value σε property
